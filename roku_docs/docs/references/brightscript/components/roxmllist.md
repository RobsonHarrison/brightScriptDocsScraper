# roXMLList
Contains a list of roXML objects.
Normally roXMLList objects are not created via CreateObject(), but are returned from various ifXMLElement functions such as GetChildElements() and GetBod
## Supported interfaces
  * [ifList](https://developer.roku.com/docs/references/brightscript/interfaces/iflist.md "ifList")
  * [ifXMLList](https://developer.roku.com/docs/references/brightscript/interfaces/ifxmllist.md "ifXMLList")
  * [ifListToArray](https://developer.roku.com/docs/references/brightscript/interfaces/iflisttoarray.md "ifListToArray")
